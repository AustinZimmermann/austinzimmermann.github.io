44022 EVO XL ROBOT

Allez-y, EVO ! Après la destruction de votre robot par les bêtes, comme un véritable héros vous avez rebondi et construit le robot XL ultime pour aider les autres héros dans le combat. Les nouvelles armes que vous avez créées, plus le fusil, les missiles et le sommet rotatif du robot vous permettront d’attaquer les bêtes venant de tous les côtés. Vous disposez de toute la technologie nécessaire, alors allez-y et gagnez la bataille d’Antropolis City pour ses courageux citoyens une bonne fois pour toutes !
