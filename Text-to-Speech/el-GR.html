Δοκιμή Κειμένου σε Ομιλία (SSML) για τη Νέα Ελληνική Γλώσσα

Προσθήκη SSML Tags:

Σημείωση: Το Web Speech API των προγραμμάτων περιήγησης έχει περιορισμένη υποστήριξη για πλήρη επεξεργασία SSML tags. Αυτή η σελίδα αφαιρεί αυτόματα τα tags αν το πρόγραμμα περιήγησής σας δεν τα υποστηρίζει εγγενώς, ώστε να ακουστεί καθαρά το ελληνικό κείμενο. Για πλήρη συμμόρφωση SSML, συνιστάται η χρήση Cloud APIs (Azure, Google Cloud, AWS).
