Hermes Synthesizer Guide & Tag Specifications

Harness advanced prosody tags to customize expressions in Greek TTS streams.

Tag Integration Engine

Since raw LLM TTS synthesizers do not naturally parse old W3C SSML protocols, Hermes compiles tag enclosures (like <excited>) into detailed style-driven prompts and programmatically stitches segments.

Perfect Pauses (<pause>)

Instead of relying on LLM interpretations of pauses, the compiler maps the time="..." attribute inside a <pause/> tag and generates custom zero-amplitude PCM arrays at exactly 24kHz.

Multi-Speaker Architecture

Combine multiple voice models by using native settings to discover premium accents. All compiled sound streams are combined locally without losing fidelity.

Tag Attribute Glossary

Tag Code Style Instruction Passed to Gemini Example usage
<excited>...</excited> Exuberant tone, elevated pitch, high-energy punctuation. <excited>Καλώς ορίσατε!</excited>
<whisper>...</whisper> Quiet, mysterious breathy tone, low vocal amplitude. <whisper>Είναι ένα μυστικό.</whisper>
<serious>...</serious> Formal, solemn, professional tone with deep register focus. <serious>Ακούστε με προσοχή.</serious>
<cheerful>...</cheerful> Warm, smiling vocal tone, friendly pacing and light cadence. <cheerful>Καλημέρα σας φίλοι μου!</cheerful>
<slow>...</slow> Sustained syllable length, steady cadence, crisp consonants. <slow>Βήμα προς βήμα.</slow>
<pause time="X"/> Injects true, non-vocal audio silence for specified duration. <pause time="1.5s"/>
