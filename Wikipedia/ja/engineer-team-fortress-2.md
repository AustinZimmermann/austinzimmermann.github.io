# エンジニア (Team Fortress 2)

<img src="http://vignette3.wikia.nocookie.net/teamfortress/images/d/d8/Engineer.png" height="250px">

**エンジニア**（英：Engineer、本名：**デル・コナガー** / Dell Conagher）は、Valve Corporation（バルブ コーポレーション）が開発したマルチプレイヤーファーストパーソン・シューティングゲーム『Team Fortress 2』（チーム フォートレス ツー、TF2）に登場する、9つのプレイアブルクラスのうちの1つである。

チームの防衛と支援を担うクラスであり、様々な機械（建造物）を組み立てることで自チームを有利に導く。

エンジニアは、テキサス出身の気さくで穏やかな性格をした労働者である。ビールとバーベキュー、そしてギターを愛する典型的な南部男のように見えるが、実際には**11の博士号（Ph.D.）**を持つ極めて優秀な天才発明家である。

戦場では前線で直接撃ち合うよりも、後方や要所で味方を支援・防衛するための拠点を構築することを主な任務とする。彼の設計図は、彼の祖父であるラジガン・コナガー (Radigan Conagher) から受け継いだものである。

## ゲームプレイ
エンジニアの最大の特徴は、専用のリソースである「**メタル（Metal）**」を消費して、戦場に様々な建造物（Buildings）を設置・アップグレードできる点にある。初期状態のメタル最大所持量は200であり、弾薬箱を拾ったり、ディスペンサーを利用したりすることで補充できる。

### 建造物 (Buildings)
エンジニアはPDA（建造ツール）を使用して、以下の4種類の建造物を設置することができる。レンチで叩くことで建造速度を早めたり、ヘルスを修理したり、メタルを消費してレベルを最大3までアップグレードすることが可能。

1. **セントリーガン (Sentry Gun)**
* 自動で敵を捕捉し、射撃を行う固定砲台。防衛の要となる。
* レベルが上がるごとに火力と連射速度が増し、レベル3ではロケット弾を発射するようになる。

2. **ディスペンサー (Dispenser)**
* 周囲にいる味方のヘルスと弾薬（およびエンジニアのメタル）を無限に補給する装置。
* 前線の維持に不可欠であり、チーム全体の生存率を大幅に引き上げる。

3. **テレポーター入り口 (Teleporter Entrance)**
4. **テレポーター出口 (Teleporter Exit)**
* スポーン地点（リスポーン地点）と前線をつなぐ転送装置。
* 足の遅いクラス（ヘビーなど）を素早く前線へ送り込むことができ、チームの攻撃の要となる。

### 基本武装
* **メイン武器**: ショットガン (Shotgun) / フロンティア・ジャスティス (Frontier Justice) / レスキュー・レンジャー (Rescue Ranger) など
* **サブ武器**: ピストル (Pistol) / ラングラー (Wrangler) / ショートサーキット (Short Circuit) など
* **近接武器**: レンチ (Wrench) / ジャグ (The Jag) / ガンスリンガー (Gunslinger) など
* ※近接武器は敵への攻撃だけでなく、建造物の修理・アップグレード、スパイのサッパー（電子機器破壊装置）の除去にも使用される。

## 開発とデザイン
エンジニアのデザインは、1996年の初代『Team Fortress』および『Team Fortress Classic』に登場した同名のクラスがベースとなっている。初代のころから「セントリーガンやディスペンサーを構築する防衛クラス」というコンセプトは確立されていた。

『Team Fortress 2』におけるキャラクターデザインは、安全ヘルメット（ハード帽）、溶接用のゴーグル、オーバーオールという、分かりやすいブルーカラーの労働者の服装を採用している。

プロモーションビデオ『Meet the Engineer』では、焚き火のそばでトラックに座り、アコースティックギターを弾きながらのんびりと語る姿が描かれている。その背後では、彼が設置したセントリーガンが敵チームを次々と蜂の巣にしているという、彼のキャラクター性を象徴するブラックユーモア溢れる映像となっている。
